How to create an exterior lab and exterior/gone users :

Only an admin is allowed to build a new exterior lab and new exterior user(s).

  1. Create the lab :

    2. Use http://biorepo.epfl.ch/biorepo/create_ext_lab : wget --post-data "mail=admin.biorepo@epfl.ch&key=xxxxxxxxxxx&lab_name=bbcf" http://biorepo.epfl.ch/biorepo/create_ext_lab

  2. Create the user(s):

    3. Use http://biorepo.epfl.ch/biorepo/create_gone_user : wget --post-data "key=xxxxxxxxxxxxxxxxxxxxx&mail=beta.testeur@epfl.ch&lab_id=lab_id&firstname=jean-michel&name=michel&user_mail=michel@epfl.ch »

  3. Create the lab’s .ini conf file

    4. add the NameOfTheLab.ini file in biorepo/conf/. NameOfTheLab must be exactly the same than the lab name chosen in step 1)
